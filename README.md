# vsl-verify-file-hashes
Calculates and compares the cryptographic hash of a file against a known or provided hash, to verify file integrity. - Focused on Lightweight utilities for scanning local project dependencies and code for common vulnerabilities by parsing dependency files (requirements.txt, package.json) or searching known vulnerable code patterns within project files.

## Install
`git clone https://github.com/ShadowStrikeHQ/vsl-verify-file-hashes`

## Usage
`./vsl-verify-file-hashes [params]`

## Parameters
- `-h`: Show help message and exit
